Tugas Mandiri 2

Buat 3 halaman website statis (Home, About, Contact) dan hubungkan tiap halaman menggunakan navigation menu. Buat tampilan sederhana namun menarik dengan CSS.

Lihat About Petunjuk Pengumpulan

Tujuan

  • Mahasiswa dapat membuat website lebih dari 1 halaman dan mempercantik tampilan dengan CSS.
  • Mahasiswa dapat menghubungkan setiap halaman website yang berbeda menggunakan navigation menu.

Deskripsi Tugas

  1. Mahasiswa membuat 3 halaman Website Statis:
    • Home — halaman utama
    • About — tentang diri / tugas
    • Contact — informasi kontak & petunjuk pengumpulan
  2. Buat tampilan dan content website sederhana namun menarik untuk dilihat atau dibaca.
  3. Hubungkan setiap page menggunakan Navigation Menu.

Catatan teknis singkat

Halaman ini statis (HTML + CSS). JavaScript kecil hanya untuk toggle menu pada perangkat mobile. Anda tidak perlu backend.
